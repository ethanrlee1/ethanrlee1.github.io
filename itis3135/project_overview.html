5. Page Design

  • Home Page (index.html) – Introduction and slogan.
  • About Us – Info about the club.
  • Events – Displays photos of events; Tells users what club activties they perform.
  • Contact Page – Form for inquiries. Links.
  • Join Page – Shows how to help or join. Dropdowns and buttons to show/hide sections.

6. Dynamic Functionality

  • Form Validation – Contact page form will use JavaScript for required fields.
  • jQuery UI Accordion – FAQ or Get Involved page will use accordion layout for answers.
  • AJAX Content Loading – Some content (e.g. announcements) will load via AJAX without reloading.
