5. Page Design

  • Home Page (index.html) – Introduction and mission; general users; info and navigation; no forms.
  • About Us – Info about the people/project; users = visitors; static content.
  • Events or Gallery – Displays photos/events; users = public; includes interactive gallery.
  • Contact Page – Form for inquiries; users = visitors; requires form validation (name/email required).
  • Get Involved – Shows how to help or join; users = public; dropdowns and buttons to show/hide sections.

6. Dynamic Functionality

  • Form Validation – Contact page form will use JavaScript for required fields.
  • Interactive Gallery – Gallery page will feature lightbox-style image viewing.
  • jQuery UI Accordion – FAQ or Get Involved page will use accordion layout for answers.
  • AJAX Content Loading – Some content (e.g. announcements) will load via AJAX without reloading.
